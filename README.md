To install typescript and rollup globally 
``` 
npm install -g typescript rollup
```

To install packages, from root directory type
```
npm i 
```

To build the main.js for the project simply type
```
npm run build
```

To run unit tests
```
npm run test
```

Transpile typescript
```
tsc
tsc -w
```
